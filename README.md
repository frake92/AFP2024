# ShadowWeb

Egy egyszerűen használható weboldal, ahol mindenféle airsoft fegyvereket lehet venni.
A létrehozásához az alábbi technológiákat alkalmazzuk:

+ HTML és CSS
+ JavaScript
+ PHP / MySQL 
+ XAMPP / PHPMyAdmin
